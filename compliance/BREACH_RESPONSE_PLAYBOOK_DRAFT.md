# Personal Data Breach Response Playbook (Draft)

**Owner:** 
**Last updated:** 2026-02-24

## 1. What is a personal data breach
Any incident leading to accidental or unlawful destruction, loss, alteration, unauthorized disclosure of, or access to personal data.

## 2. Immediate actions (first 0–4 hours)
- Contain the incident (disable keys, rotate credentials, block traffic, restrict access)
- Preserve evidence (logs, timestamps)
- Assign incident lead and open an incident channel

## 3. Assessment (within 24 hours)
- What happened? (timeline)
- Systems affected
- Data affected (types, volume)
- Users affected (who/where)
- Risk assessment (harm likelihood)

## 4. Notifications
- Internal stakeholders
- Users (if required)
- Rwanda DPO breach report (use official form)

## 5. Official DPO breach form
DPO provides a breach report form:
- https://dpo.gov.rw/fileadmin/DPO/Services/Personal-Data-Breach-Report-Form.pdf

Recommended internal prep before filling:
- incident summary
- categories of data affected
- number of data subjects affected
- mitigation measures taken
- contact person

## 6. Post-incident
- Root cause analysis
- Security improvements
- Document lessons learned
