# Cross-Border Storage/Transfer Authorization Checklist (Rwanda DPO) — Draft

Use this checklist to prepare submissions for:
- Authorization to **Store Outside Rwanda**
- Authorization to **Transfer Outside Rwanda**

## 1. Confirm where data is stored/processed
- Database region:
- Storage bucket region:
- Backups region:
- Email provider region:
- Analytics provider region:

## 2. Identify what personal data is involved
- Account identifiers (name/email/phone)
- User-generated content (reviews, listings, event photos)
- Logs and IPs
- Payment/transaction records

## 3. Describe safeguards
- Encryption in transit
- Access controls / least privilege
- Audit logging
- Vendor DPAs
- SCCs (DPO SCC PDF)

## 4. Collect attachments
- System architecture overview (1–2 pages)
- Vendor list + regions
- Security controls summary
- SCC pack (as applicable)

## 5. DPO forms
- External transfer authorization form:
  - https://dpo.gov.rw/fileadmin/DPO/Services/application-form-for-external-transfer-authorization.pdf
- External storage authorization form:
  - https://dpo.gov.rw/fileadmin/DPO/Services/application-form-for-external-storage-authorization.pdf
