## 5. Reasons for storing personal data outside Rwanda
We request authorization to store personal data outside Rwanda due to:
- Use of reputable managed cloud providers with robust security controls and reliability
- Availability of required infrastructure, scalability, and operational resilience
- Encryption-at-rest and encrypted backups offered by vendors
- Disaster recovery and backup capabilities to ensure continuity of service

## 6. Safeguards and security measures
We implement technical and organizational measures including:
- Role-based access controls for administrative access
- Least-privilege access model
- Encrypted transport (TLS/HTTPS)
- Secrets management for credentials and API keys
- Incident response process and breach reporting readiness

Additionally, our service providers implement security controls documented in their contractual and security documentation (attached). For example, Supabase’s DPA describes encryption at rest, encrypted backups, access controls, logging, monitoring, and security incident notification.
